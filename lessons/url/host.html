The host name is arguably the most important piece of a URL. The host name informs you which website you are on and is often times the name you use when referring to a site. Think of Twitter or Instagram or Facebook or Youtube or Wikipedia or Twitch, we refer to these sites by a shortened version of their host name. When we casually speak of Youtube in a conversation we all know we are talking about www.youtube.com

Let's take a look at the Wikipedia example host name:

www
wikipedia
org

What may stand out is that the host name is itself comprised of three even smaller parts, each separated by a period.

1) Subdomain

www.wikipedia.org

This part of the host name can be anything at any point. It is at the discretion of whomever controls the website to choose whatever they would like for this piece of the host name. Often times it is used for organizing which version of a webpage you are seeing. Most commonly the default will be www as a shorthand for World Wide Web. Wikipedia was chosen as an example because it displays an interesting usage of the subdomain. Click the buttons below to see what happens when you change the subdomain of Wikipedia.

en.wikipedia.org/wiki/url

2) Top Level Domain

www.wikipedia.org

The top level domain was originally intended to represent the purpose of a website, like .edu being used for school websites. Now however the TLD can be used to decorate a websites host name. There is a finite number of choices for a TLD controlled by a governing body called the Internet Corporation for Assigned Names and Numbers. To see all possible options visit this officially-maintained, extensive list.

3) Domain Name

www.wikipedia.org

The domain name is a bit trickier because it is the combination of the middle part and the previously discussed TLD. This combination serves as the name of a website and must be unique. Only one website may exist with a given domain name.

Try making your own domain name. Visit a domain name registrar to see if you can find an available domain name that you like. Remember you can only use TLDs from the approved list above. Type your desired name in the search box and see what comes out.
