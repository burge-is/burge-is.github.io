The host name is arguably the most important piece of a URL. The host name informs you which website you are on and is often times the name you use when referring to a site. Think of Twitter or Instagram or Facebook or Youtube or Wikipedia or Twitch, we refer to these sites by a shortened version of their host name. When we casually speak of Youtube in a conversation we all know we are talking about www.youtube.com

Let's take a look at the Wikipedia example host name:

www
wikipedia
org

What may stand out is that the host name is itself comprised of three even smaller parts, each separated by a period.

1)Sub Domain

www.wikipedia.org

This part of the host name can be anything at any point. It is at the discretion of whomever controls the website to choose whatever they would like for this piece of the host name. Often times it is used for organizing which version of a webpage you are seeing. Wikipedia was chosen as an example because it displays an interesting usage of the subdomain.
